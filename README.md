# SMS Coupon Manager

A tool for creating and validating SMS (Store Management Suite) coupons.

## Quick Start

1. Download `SMSCouponManager.bat` and `sms_loader.py`
2. Put both files in the same folder
3. Double-click `SMSCouponManager.bat`

The application will automatically download and run.

## Requirements

- Windows 10/11
- Internet connection (for first download)
- Python will be installed automatically if not present

## Features

- Create all types of SMS coupons (Dollar Off, Percent Off, BOGO, Bundle, etc.)
- Validate existing coupons against SMS rules
- Auto-detect SMS database connection
- Configure raffle rules on receipts
